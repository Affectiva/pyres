Introduction¶

PyRes is a resque clone built in python. Resque is used by Github as their message queue. Read the blog post from github about how they use resque in production.

synopsis:Any job which takes a little while to run can be put on a message

queue. Read our Example implementation of how a PyRes can be used to spam check comments.
